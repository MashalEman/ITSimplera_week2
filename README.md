## 📋 Project Overview
This project covers two core Network Administration concepts:

1. **IP Addressing** – Designing subnets using Fixed Length Subnet Masking (FLSM) and Variable Length Subnet Masking (VLSM).
2. **VLAN Configuration** – Implementing and verifying VLANs, access ports, and trunk links on Cisco switches using Cisco Packet Tracer and GNS3.

## 🎯 Objectives
- Understand the concepts and importance of FLSM in network design.
- Learn VLSM to efficiently allocate IP addresses based on host requirements.
- Calculate network addresses, subnet masks, broadcast addresses, and usable host ranges.
- Understand the concept, purpose, and advantages of VLANs.
- Configure and verify VLANs on switches using Cisco Packet Tracer & GNS3.

## 🛠️ Tools Used

- PuTTY

## 🔧 Implementation Summary
- Designed IP addressing schemes using both FLSM and VLSM methods for different subnet size requirements.
- Built a network topology in Cisco Packet Tracer with multiple VLANs, access ports, and trunk links between switches.
- Recreated the same topology in GNS3 for cross-platform verification.
- Verified configuration using Cisco IOS commands:
  - `show vlan brief`
  - `show interfaces trunk`
  - Ping tests between VLANs to confirm connectivity

## ✅ Key Learning Outcomes
- Clear understanding of the differences between FLSM and VLSM.
- Practical experience calculating subnet masks, broadcast addresses, and host ranges.
- Hands-on skills in configuring and troubleshooting VLANs on Cisco switches.
- Improved documentation and professional reporting skills for enterprise networking tasks.
